AutoHotkey


; 腳本循環範本。
#MaxThreadsPerHotkey 2  ; 設定為 2，這樣正在循環時按第二次按鍵才有反應

`::
    ; 這裡的 Toggle 就像是一個開關變數
    Toggle := !Toggle 
    
    ; 只要 Toggle 是「真」(1)，就會一直循環
    while (Toggle) {
            ; 此區塊為腳本執行內容。


        
        ; 【關鍵】每次循環結束會回頭檢查 while (Toggle)
        ; 如果你剛才按了第二次按鍵，Toggle 變成了 0 (假)，循環就會自動停止
    }
return
