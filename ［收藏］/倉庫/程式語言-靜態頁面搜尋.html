JavaScript


document.getElementById('searchInput').addEventListener('keyup', function() {
  var filter = this.value.toLowerCase();
  var lis = document.querySelectorAll('#搜尋 tr');
  lis.forEach(function(tr) {
    tr.style.display = tr.textContent.toLowerCase().includes(filter) ? '' : 'none';
  });
});

    AI分析講解
1. document.getElementById('searchInput').addEventListener('keyup', function() { ... });
用途/功能：
取得網頁上 id 為 searchInput 的輸入框（通常是 [input]）。
為這個輸入框綁定 keyup 事件監聽器，當使用者每次在輸入框按下按鍵並釋放時，就會觸發後面的匿名函式。
說明：
這是即時偵測使用者輸入內容的常見做法，便於實現「邊輸入邊搜尋」的效果。
2. var filter = this.value.toLowerCase();
用途/功能：
取得使用者在輸入框內輸入的文字。
將輸入文字轉為小寫（toLowerCase()），以實現不區分大小寫的搜尋。
說明：
這樣使用者輸入「abc」或「ABC」都能正確比對，提升使用體驗。
3. var lis = document.querySelectorAll('#搜尋 tr');
用途/功能：
取得 id 為 搜尋 的元素（通常是 [table id="搜尋"]）下的所有 （表格列）。
說明：
這裡假設你的表格的 id 是「搜尋」，而你想要根據輸入內容來顯示或隱藏每一列（row）。
4. lis.forEach(function(tr) { ... });
用途/功能：
針對每一個  執行一個匿名函式，進行逐列檢查。
說明：
這樣可以針對表格中的每一列進行個別操作。
5. tr.style.display = tr.textContent.toLowerCase().includes(filter) ? '' : 'none';
用途/功能：
取得每一列的所有文字內容（textContent），轉為小寫。
檢查這些內容是否包含使用者輸入的關鍵字（includes(filter)）。
如果包含，則顯示這一列（display = ''，即預設顯示）。
如果不包含，則隱藏這一列（display = 'none'）。
說明：
這樣當使用者輸入關鍵字時，表格只會顯示符合搜尋條件的資料列，其他的都會被隱藏。
這行程式碼裡的 「? :」 是 JavaScript 的三元運算子（Ternary Operator），它是一種條件判斷的簡寫語法。
語法格式如下：條件判斷 ? 條件為真時執行的表達式 : 條件為假時執行的表達式
等價於：
if (tr.textContent.toLowerCase().includes(filter)) {
    tr.style.display = '';
} else {
    tr.style.display = 'none';
}
總結
這段程式碼讓你可以在一個有 id="搜尋" 的表格中，使用一個 id="searchInput" 的輸入框即時過濾表格內容。使用者每次輸入時，表格會自動只顯示包含搜尋關鍵字的列，達到快速搜尋的功能。
